NOTES:
- When loading .txt.gz matrix files, they are tab-separated with gene/CpG labels in column 1.
  Use: pd.read_csv(path, sep='\t', compression='gzip', index_col=0).
- Methylation_Group column values include underscores in MOESM4 ("MPNST_G1", not "MPNST-G1"); the paper text uses hyphens.
- The Clinical Data table has a small number of rows that are blanks/metadata; use Tumor_ID non-null to filter to the 108 samples.
- Recurrence is coded 0 (no event) or 1 (event); Time to Recurrence or Last Follow Up is in YEARS (not days).
- The IDATs directory and matrix files together let R packages (minfi, ChAMP) be re-run if needed, but most
  downstream analyses can be done directly on GSE207207_average_beta_values.txt.gz.
- Tumor_Sample_Barcode in MOESM5 MAF matches Tumor_ID/WES label in MOESM4 (e.g., "M1014").
- Sample labels: M* prefix = MPNST/atypical samples, NF* = neurofibroma (plexiform or atypical) samples,
  cNF*/Cut* = cutaneous neurofibromas.
